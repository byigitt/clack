# Bundled soundpacks

These soundpacks ship with clack so it works out of the box. They originate from
the [thock-soundpacks](https://github.com/kamillobinski/thock-soundpacks)
collection and upstream sources.

## Licenses

- Most packs are **MIT** licensed, sourced from
  [tplai/kbsim](https://github.com/tplai/kbsim) and the
  [Mechvibes](https://github.com/hainguyents13/mechvibes) community. Each pack's
  `config.json` records its own `license` and original author.
- **Quack** uses the **Pixabay Content License**.

Proprietary packs are intentionally **not** bundled.

Each `<uuid>/config.json` keeps the original `metadata.author` and `license`
fields — see those for per-pack attribution. To add your own packs, drop a pack
folder into `~/Library/Application Support/Clack/Soundpacks/` and relaunch.
